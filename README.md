# Online .txt Reader / 在线小说阅读

## 功能

根据txt小说(UTF-8)，自动分章节，根据访问IP记录当前阅读进度。例子：1000.txt/《朝花夕拾》。

章节筛选：utils.php => _filterRegexp

前进后退切换：左右方向键 or 触屏左右滑动

## 实现

- PHP(enable extension sqlite3)
- Materialize UI
- Ajax/Pjax
- Vue.js
